⏰ LitterBox (Temporary File Hosting) FAQ

What is LitterBox?

LitterBox is Glowi.es' temporary file hosting service inspired by catbox.moe. Upload files that automatically delete after a set time period - perfect for quick file sharing without permanent storage.

How much can I upload?

LitterBox allows files up to 1GB per upload - 5x larger than our permanent Images storage. Perfect for large files, video clips, or bulk uploads that you don't need to keep forever.

What are the expiration options?

You can choose from: • 1 Hour - for very temporary shares • 12 Hours - for same-day file sharing • 1 Day - for quick access period • 3 Days - for short-term collaboration

Are files encrypted?

Yes! Like our permanent Images storage, LitterBox files are also encrypted end-to-end with AES-256-GCM encryption. Your files are secure even during the temporary hosting period.

Can I password protect LitterBox uploads?

Absolutely! Add a password to any LitterBox file for extra security. Recipients must enter the correct password to download the file.

What happens after the file expires?

Once the expiration time passes, the file is automatically and permanently deleted from our servers. Nobody can access it anymore - not even you.

How long is the share code?

LitterBox files get a random 16-character code (e.g., ab1cd2ef3gh4ij5k) compared to the 6-character codes for permanent Images. This ensures uniqueness across millions of temporary uploads.

Can I delete files early?

Yes! You can manually delete any LitterBox file before its expiration time. Once deleted, it's immediately removed for everyone.

Are there file type restrictions?

Same as Images: Executable files (.exe, .scr, .cpl), Word documents (.doc, .docx, .docm), and Java files (.jar) are blocked. All other file types are supported.

Can I use LitterBox for backups?

Not recommended - LitterBox is designed for temporary sharing, not long-term backup. Use our permanent Images storage for files you want to keep indefinitely.

What if I need the file after it expires?

Once a file expires, it cannot be recovered. For permanent storage with no expiration risk, use our Images section instead.
