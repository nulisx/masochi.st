What is E2EE File Hosting?

End-to-End Encryption (E2EE) means your files are encrypted on your device before being sent to our servers. Only you and anyone you share the link with can decrypt and view the files. Even our servers cannot see the contents of your files.

How much storage do I get?

You can upload files up to 200MB each. The total storage capacity depends on your account tier. Each file gets a unique 6-character code for sharing.

Can I password protect my files?

Yes! When uploading, you can set an optional password that recipients must enter to download the file. This adds an extra layer of security beyond the encryption.

Can files expire?

Yes! You can set an expiration date when uploading. After the specified date, the file will no longer be accessible to anyone, not even you.

What file types are blocked?

Executable files (.exe, .scr, .cpl), Word documents (.doc, .docx, .docm), and Java files (.jar) are blocked for security reasons. All other file types are allowed.

How do I share a file?

After uploading, you'll receive a unique link (e.g., glowi.es/file/abc123). Share this link with anyone. If you password-protected it, they'll need to enter the password to download.

Is my encryption key stored?

Your encryption key is generated client-side and sent with the download link. We never store or see your encryption keys, ensuring maximum privacy.

Can I delete my files?

Yes! You can delete any file from your "Images" section anytime. Once deleted, the file is permanently removed and no longer accessible.

What about the file download count?

We track how many times each file has been downloaded so you can see how popular your shared content is.

Is there a size limit?

Individual files can be up to 200MB. For larger files or bulk uploads, consider using our LitterBox temporary storage feature.
